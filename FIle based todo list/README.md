# Python To-Do List App

A simple **Python-based To-Do List** application that allows users to **add, remove, and view tasks** with optional password protection. This project is perfect for beginners learning Python, file handling, and basic programming concepts.

---

## Features

- **Add Tasks:** Users can add new tasks to the list.
- **Remove Tasks:** Remove tasks by name from the list.
- **View Tasks:** Display all tasks in a clean numbered format.
- **Password Protection (Optional):** Protect your tasks with a password before viewing.
- **File Storage:** Tasks are stored in a local text file (`task.txt`) for persistence.

# Random Password Generator (Python)

## Overview
This is a simple **command-line random password generator** built using Python.  
The project was created as part of my learning phase to strengthen fundamentals such as functions, loops, randomness, and basic input validation.

The program generates a random password based on a user-defined length within a safe range.

---

## How It Works
1. The user chooses whether to generate a password.
2. The user inputs the desired password length.
3. The program randomly selects characters and builds the password.
4. The generated password is displayed in the terminal.
